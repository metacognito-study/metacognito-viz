Photoelectron Spectroscopy (PES) — read an electron configuration off the peaks

Each peak = one occupied subshell. Taller peak → more electrons in that subshell; further left → more tightly bound (closer to the nucleus, core electrons). x = binding energy, MJ/mol (plotted high → low) y ∝ number of electrons
